## 如何添加真实 Emoji 图像

### 方法 1: 使用 Twemoji (推荐)
1. 下载 Twemoji: https://github.com/twitter/twemoji
2. 准备 16x16 PNG 图像
3. 使用图像编辑工具批量填充到纹理的 Emoji 区域

### 方法 2: 使用 Noto Emoji
1. 下载 Noto Emoji: https://github.com/googlefonts/noto-emoji
2. 缩放到 16x16 像素
3. 填充到对应位置

### 方法 3: 从系统字体渲染 (需要额外工具)
- macOS: Apple Color Emoji
- Linux: Noto Color Emoji
- Windows: Segoe UI Emoji
