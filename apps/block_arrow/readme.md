# 总体需求
我想做一个新的游戏:
- 给定一张比较简单的像素画（例如分辨率8x8）
- 用一些俄罗斯方块形状（包含经典块和一些扩展块）随机组合，把像素画内非背景的像素全覆盖
- 然后在方块上，设置四个方向的箭头（上下左右）
- 玩家点击箭头如果没有遮挡，方块就可以飞走，把所有方块点飞，则通关
- 帮我想想关卡如何程序自动设计，给定像素画

# 扩展思考
- 经典 tetromino：I,O,T,S,Z,J,L（面积 4）
- 扩展块：建议加入
  - triomino（3 格）：I3, L3
  - domino（2 格）
  - monomino（1 格）谨慎：只在无法整除或为保证可铺时使用
  - pentomino（5 格）：会显著增加搜索空间，但 8x8 小盘仍可做
  - 每个块：用相对坐标集合表示，例如 [(0,0),(1,0),(0,1)]，并预生成旋转/翻转形态。
- 可以参考apps/colorblk的块定义
